Overview

Features

  • Wikibase constitutes a powerful, flexible and customizable knowledge base
  • It provides MediaWiki as its front end and a rich JavaScript interface for data management
  • Its data model places a high priority on localization and knowledge diversity
  • Robust data export options include JSON, RDF/XML, N3, YAML and more
  • Powerful local and federated SPARQL queries (with updates on changes) extend the potential available data from a single node to thousands
